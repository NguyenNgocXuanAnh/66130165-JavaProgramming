## 🔎 Nội dung học tập
- Các khái niệm nền tảng: biến, toán tử, câu lệnh điều khiển, vòng lặp

- Nhập/xuất dữ liệu và tương tác với người dùng

- Lập trình hướng đối tượng (OOP): class, object, constructor, encapsulation

- Làm việc với stream và xử lý file

## 🔥 Mục tiêu

- Xây dựng nền tảng vững chắc về ngôn ngữ lập trình Java từ cơ bản đến nâng cao.

- Phát triển tư duy logic, khả năng phân tích và giải quyết bài toán trong lập trình.

- Nắm vững lập trình hướng đối tượng (OOP) để áp dụng vào các dự án thực tế.

- Làm quen với việc xử lý dữ liệu, làm việc với file và stream trong Java.

*Note:💡Tư duy như một lập trình viên thực thụ. Không chỉ học để biết, mà còn có thể tự tin viết và phát triển các chương trình Java một cách độc lập.*
